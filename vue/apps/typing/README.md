# Vue Typing App

A typing app implementation with Vue.

## Requirements

- Yarn

## How to

### Start the development version

`yarn && yarn run serve`

### Start the build version:

1. `yarn && yarn run build`
2. (`yarn global add serve`)
3. `serve -s dist`
