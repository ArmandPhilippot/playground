# React Redux ToDos

A ToDo app implementation with React and Redux.

## Description

This a simple ToDo application used to learn Redux.

## Requirements

- Yarn

## How to

### Start the development version

`yarn run start`

### Start the build version:

1. `yarn run build`
2. (`yarn global add serve`)
3. `serve -s build`
